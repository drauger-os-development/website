System Requirements
Minimum Hardware Requirements
Hardware under these requirements may still run, but the experience will be so poor that we STRONGLY recommend against it, and in some cases, may have placed limiters in our installer in order to prevent installations that will fail based on internal testing.
CPU Dual-core, 1.8 GHz, 64-bit processor
RAM 4 GB
Storage 32 GB (aprox. half used once installation complete)
GPU Integrated Graphics
Screen Resolution 1024 x 768 at 30 Hz
External Ports 1 Display output (HDMI, DisplayPort, USB-C, VGA, DVI, etc.) 3 USB 2.0 Ports (1x for bootable USB drive, 1x for keyboard, 1x for mouse)
Recommended Hardware Requirements
CPU Quad-core, 2 GHz, 64-bit processor
RAM 8 GB DDR4 3200 MHz
Storage 256 GB NVMe SSD
GPU Nvidia GTX 1050, AMD Radeon RX 550, or equivalent
Screen Resolution 1080p at 60 Hz
External Ports 1 Display output (HDMI, DisplayPort, USB-C, VGA, DVI, etc.) 2 USB 2.0 Ports (1x for keyboard, 1x for mouse) 1 USB 3.0 Port (1x for bootable USB drive)
Best Possible Results
CPU 12-core, 4+ GHz, 64-bit processor
RAM 8 GB DDR5 5000+ MHz
Storage 500 GB NVMe 4.0 SSD
GPU AMD Radeon RX 6800 or better AMD card
Screen Resolution 1080p at 120 Hz
External Ports 1 Display output (HDMI, DisplayPort, USB-C, VGA, DVI, etc.) 2 USB 2.0 Ports (1x for keyboard, 1x for mouse) 1 Thunderbolt 4 Port (1x for bootable USB drive)
In our testing, anything past these specs sees diminishing returns with our installer.
Performance per game depends on that particular game.
